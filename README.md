# Exercício Guiado de Git – Gestão da Configuração de Software

Este repositório foi preparado para uma aula prática sobre:
- controle de versões;
- branches;
- commits organizados;
- baseline;
- rastreabilidade.

## Objetivo
Aplicar uma pequena melhoria em um sistema legado simples, organizando o trabalho com Git.

## Estrutura
- `app.py`: código legado propositalmente simples, com alguns problemas de manutenibilidade

## Como executar o sistema
```bash
python app.py
```
